# Intro-agent

Intro helps events turn random networking into intentional connection, and proves the
value of that connection to the people who pay for it.

## Positioning

Intro is positioned differently for each stakeholder because each one is buying a
different outcome:

- **Attendees** get better relationships.
- **Sponsors** get measurable ROI.
- **Organizers** get recurring revenue through happier attendees and renewing sponsors.

See [`docs/positioning.md`](docs/positioning.md) for the full stakeholder positioning,
including the problems each stakeholder has, how to describe Intro to them, their
benefits, and the sponsor-to-organizer demand relationship.
